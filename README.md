## Installatie

1. Clone het repository:

   ```bash

   cd Project-experimenteren2
   ```

## 📁 Project Structuur

```
Project-experimenteren2/
├── README.md           # Project documentatie
├── Data                # Hier komt de data van ons experiment (resultaat en verwerking)
├── docs/               # Aanvullende documentatie vooral als voorbereiding op de practica
└──  Source-code/       # Broncode die we gebruiken van de arduino en python.
```

### Workflow

1. Maak een nieuwe branch voor je feature: `git checkout -b feature/jouw-feature`
2. Commit je wijzigingen: `git commit -m 'Beschrijving van wijzigingen'`
3. Push naar de branch: `git push origin feature/jouw-feature`
4. Open een Pull Request
